# Hangman

Mo, Chester, Denzel hangman game using Node.js and express

![Denzal](.\public\assets\denw.jpg)

## Requirements

-   Install Node.js

## Usage

```
git clone <REPO>
cd Hangman
npm i
npm start
```

Exposed on localhost:3000
