# Hangman

Mo, Chester, Denzel hangman game using Node.js and express

<img src="./public/assets/denw.jpg" alt="denzel" height="200" width="200"/>

## Requirements

-   Install Node.js

## Usage

```
git clone <REPO>
cd Hangman
npm i
npm start
```

Exposed on localhost:3000
